### 1. [omega-core](https://github.com/prancer-io/ExoGenesis-Omega/tree/main/omega/crates/omega-core) - The Foundation
Core types and traits for universal intelligence. Defines the 12-tier memory system, 7 temporal loops, and multi-paradigm architecture support.

### 2. [omega-agentdb](https://github.com/prancer-io/ExoGenesis-Omega/tree/main/omega/crates/omega-agentdb) - SIMD-Powered Vector Intelligence
High-performance vector database with **13-41x SIMD acceleration**. Features HNSW indexing, Reflexion learning, Causal reasoning, and Skill management.

### 3. [omega-memory](https://github.com/prancer-io/ExoGenesis-Omega/tree/main/omega/crates/omega-memory) - Cosmic Memory Architecture
12-tier hierarchical memory system spanning from instant (1ms) to omega (universe-age) timescales, with automatic consolidation and semantic retrieval.

### 4. [omega-loops](https://github.com/prancer-io/ExoGenesis-Omega/tree/main/omega/crates/omega-loops) - Temporal Cognition
Seven nested feedback loops operating from milliseconds to decades, enabling multi-scale learning, adaptation, and evolution with emergent pattern detection.

### 5. [omega-meta-sona](https://github.com/prancer-io/ExoGenesis-Omega/tree/main/omega/crates/omega-meta-sona) - Self-Organizing Architecture
Intelligence design engine using Monte Carlo Tree Search and Proximal Policy Optimization for automated architecture discovery and evolution.

### 6. [omega-runtime](https://github.com/prancer-io/ExoGenesis-Omega/tree/main/omega/crates/omega-runtime) - Production Orchestration
Enterprise-grade runtime with health monitoring, circuit breakers, retry policies, graceful degradation, and comprehensive event system.

### 7. [omega-persistence](https://github.com/prancer-io/ExoGenesis-Omega/tree/main/omega/crates/omega-persistence) - Durable Storage
SQLite-backed persistence with ACID guarantees for all entity types, indexed queries, and backup support.
